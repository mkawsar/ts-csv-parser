# Node.js TypeScript

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/mkawsar/ts-csv-parser
   cd ts-csv-parser
   ```
2. Install the project dependency
    ```bash
    npm install
    ```
## Usage
1. Start the Application: Run the compiled JavaScript code.
    ```bash
    npm run dev
    ```
